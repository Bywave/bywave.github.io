Grid System

Basic Information

All you need to know about the grid system. Basically this system is inspired from 960gs, bootstrap, foundation etc., so if you are familiar with those framework you will get along with this very well. It has 2 different approaches in terms of grid, the common 12 column grid, and the new ANY column grid. It is designed in Responsive Grid support to adopt 4 major devices such as Mobile, Tablet, Desktop, Desktop Wide Screen and it all named as (Mobile) light, (Tablet) moderate, (Desktop) normal, (Desktop Wide Screen) heavy.

Normal / Flexibility - first thing

The default grid, etc., would be Normal, as such matter no need for a class name "-normal".
All grids is automatic being floated with each other and All grids in Mobile is display block with each other.

FLUID

It uses percentage width in every grid, and divided into 4 breakpoints (Mobile, Tablet, Desktop, Desktop Wide Screen).
The class ".container" have diffrent width in every breakpoints. FLUID layout can achieve by putting a class name "full" in the body.

Text Alignment

Alignment of text made easy using these classes ".text-left", ".text-center", ".text-right". All the images inside of these classes will also be aligned properly. Sometimes text needed to be centered, etc. in some devices like tablets or mobile. So there are some class will help you to achieve that. Use a class name ".text-center-light", ".text-center-moderate", ".text-center-normal", ".text-center-heavy".
