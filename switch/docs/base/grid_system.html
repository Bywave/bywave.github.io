Grid System

A very flexible grid system that deals with multiple devices in default of 12 column layout.
Switch grid makes it simple for users to adopt, it uses percentage width to adopt responsive layouts. It is meant for all Devices, because it is built with Desktop and Mobile approach.



Basic Setup

To be able to use Switch Grid System, first things first, you need to include class name's ".container" followed by ".row" and ".grid-(#)" (where "#" is a number of columns).

Container

  • This element is used for setting a width of the content. Every devices such as (Mobile, Tablet, Desktop, Wide) have different breakpoints / width.
  • Container has a default style of padding to match with grid's gutters.

Row

  • This element will be the holder of all grids, to clear all the floats. Since all the grids is floated to left.
  • Row has an offset of negative margins, to accomodate all the widths of the grids, since all the grids have a default style of padding.

Grid-(#)

  • Grid-x is divided into 12 columns to make a layout, so you need to add a number for appropriate grids. So every grid will be ".grid-(#)" (where "#" is a number of columns) up to 12 columns.
  • All grids have a default style of padding that will serve as gutters of every grids.

Important!

Class name ".container" is used only when you need to set a breakpoint / width. It is primarily used as a container of all the content. No need for a container, especially when dealing in nested grids.

All grids in mobile is automatically displayed as block element. All floats will be removed.



ANY Column Grid

A versatile component that can achieve any number of columns. It's so easy to setup when you stuck on 12 column grid, just use a regular class name ".grid-(#)", but this time it needs to be put together with the ".row" class name, followed by class name ".grid".

This is a very helpful when you need a 16 column grid system, 24, etc.

Important!

Class name ".grid" is a number of columns that depends on ".grid-(#)". When you put ".grid-5", it means you need to put 5 class name ".grid".

Multiple Devices

When you deal with responsive design, sometimes you might want to have a different layout in every devices. There are some options to trigger the grids in different screens. It starts with the class name's ".small-(#)" - (Small screen / Mobile), ".medium-(#)" - (Medium screen / Tablet), and ".large-(#)" - (Large screen / Wide Screen).

Important!

For normal screen / desktop is considered as a default, because it can be categorized as part of regular ".grid-(#)".

These are only optional when you want to adopt different layout in some devices. you can still use a regular ".grid-(#)".

To simply see it in action, just resize your browser.

Grid Off

You can disable style using a class name ".grid-off". It means there is no (padding, margins, etc.).

Different styling can also be applied like the Grid Cell's options. Use a class name ".border" and ".with-bg" for further styling.

Important!

For inner content, just use a class name ".grid-content", for better spacing.

SASS Configuration Variables

// Grid Columns 
// Grid Columns - Number of columns the layout will be divided
$grid-columns: 12px !default;

// Grid Gutters
// Grid Gutters - Default Gutters
$grid-gutter-default: 20px !default;

// Container Padding - for small screen
// Sometimes, you needed to put extra padding on small screens.
$grid-container-row-small: 30px !default;

// Grid Margins for small screen, In default every grid have a margin bottom styling
$grid-margin-bottom-small: 10px !default;

// Grid Content
// Grid Content - Basic styling
$grid-content-bg: none !default;
$grid-content-color: inherit !default;

// Grid Content - padding styling, adjustable top / right / bottom / left.
$grid-content-padding-top: 20px !default;
$grid-content-padding-bottom: 20px !default;
$grid-content-padding-right: 20px !default;
$grid-content-padding-left: 20px !default;

// Grid Cell
// Grid Cell - with border in between styling.
$grid-cell-border-between: 1px solid darken($color-gray-lighten, 10%) !default; 

// Grid Cell - with background Styling
$grid-cell-bg-bg: $color-primary !default;
$grid-cell-bg-color: $color-white !default;
$grid-cell-bg-link: $color-tertiary !default;

// Grid Off
// Grid Off - with background styling. 
$grid-off-bg-bg: $color-primary !default;
$grid-off-bg-color: $color-white !default;
$grid-off-bg-link: $color-tertiary !default;

// Grid Off - with border in between styling. 
$grid-off-border-between: 1px solid darken($color-gray-lighten, 10%) !default;

See config.scss for complete reference.
