Header

A content located on top of the page. To provide a better markup and organize all necessary content in a single container. Basically, it is composed of (Top Bar and Nav Bar).



Basic Setup

Wrap all your top content in a class name ".header". This will be the holder of all necessary top content.

Important!

For floating content, use these utilities ".left" and ".right" for better arrangements and margins. These will help you to manipulate the content.

Since all floats element in mobile devices is automatically displayed as block, If you wanted to force the content into floats, just use other utilities namely ".force-left" and ".force-right".

If you will be having a multiple floating content, it is better to use a (.left or .right as your container) to make the content organized.

These classes have a default (margin-left and margin-right) respectively.



Top Bar

An extra element if you have a content on a very top of the header. Use a class name ".top-bar".

This Top Bar can also be useful for having a (social, contact infos, logos, etc.). An optional content like ".header-social" and ".top-nav".

Important!

Nav bar have a lot of activities, in regards to mobile devices. It's going to have a switch and fold function to make it work on mobile devices.

For functionality to work, you need to set up these classes ".nav-bar-content", ".nav-fold" and ".nav-switch". Also Javascript needed in this element, so it must be included.



Nav Bar Content

This will be the container of all visible content in mobile devices. Example would be the Logos, Nav Switch, etc.

For further understanding, resize your browser to see it in action.

HTML example markup
<div class="nav-bar-content">
	...
</div>


Nav Fold

This element will be hidden in mobile devices. So content like navigation, forms, etc. will be inside.

This element will only visible (mobile devies) when Nav switch is clicked.

HTML example markup
<div class="nav-fold">
	...
</div>


Nav Switch

For Nav Fold to work, you need this element, a class name ".nav-switch". This will trigger the nav fold to be visible in mobile devices.

This is what it looks like on mobile devices a three layer button line.


HTML example markup
<a class="nav-switch">
	<span class="line"></span>
	<span class="line"></span>
	<span class="line"></span>
</a>

Header Extras

Any element can be placed inside of header, including Form Elements, Buttons, etc.

Header with Form Elements

Use this element if you need some basic search box inside the header. Search box can be located either on topbar and navbar. Use a class name ".header-form", this will be the container of all necessary elements you need. For form styling see Forms.
