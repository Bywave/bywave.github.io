Blocks

Every layout have a "block" of content, this will help you to make your layout consistent in terms of spacing (margins, paddings, etc.).



Basic Setup

Use a class name ".block", this will provide you a great spacing to each content blocks.

A divider can be applied by putting a border in every blocks.

!important

This is very helpful, when you have a fluid layout with different block of content. You can easily use this to control each blocks. You can style the margin and padding on whatever you desire.

SASS Configuration Variables

// Block Container Padding - adjustable top / bottom
$block-padding-top-default: 20px !default;
$block-padding-bottom-default: 40px !default;

// Block - padding for Small screen
$block-padding-top-small: 20px !default;
$block-padding-bottom-small: 20px !default;

// Block - padding for Medium screen
$block-padding-top-medium: 20px !default;
$block-padding-bottom-medium: 20px !default;

// BLock Margin
$block-margin-bottom-default: $margin-bottom-default !default;
$block-margin-bottom-small: $margin-bottom-default !default;
$block-margin-bottom-medium: $margin-bottom-default !default;

// Block Border styling
$block-border-bottom: 1px solid $color-tertiary !default; //1px solid darken($color-gray-lighten, 10%)

// Block Content Padding - left / right
$block-content-padding-left: 20px !default;
$block-content-padding-right: 20px !default;

See config.scss for complete reference.
