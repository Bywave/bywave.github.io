Articles

An element that will provide a straight forward approach dealing with, news, post/blogs, comments or any type of informations.



Basic Setup

Start with a class name ".article" followed by ".article-content".

Important!

For article date, use ".article-date". If you need a block that will put all your related links, Use a class name ".article-links".

For Easy Layout of images with text by side, Just use Grid System to make it sematic and responsive.

For Images, please see Images for reference.

SASS Configuration Variables

// Article - margin bottom
$article-margin-bottom: $margin-bottom-default !default;

// Article Content - padding adjustable top, right, bottom, left
$article-content-padding-top-default: 0px !default;
$article-content-padding-bottom-default: 0px !default;
$article-content-padding-left-default: 0px !default;
$article-content-padding-right-default: 0px !default;

// Article Content - padding for Small screen
$article-content-padding-top-small: 10px !default;
$article-content-padding-bottom-small: 10px !default;
$article-content-padding-left-small: 10px !default;
$article-content-padding-right-small: 10px !default;

// Article Content - padding for Medium screen
$article-content-padding-top-medium: 10px !default;
$article-content-padding-bottom-medium: 10px !default;
$article-content-padding-left-medium: 10px !default;
$article-content-padding-right-medium: 10px !default;

// Article Date styling
$article-date-color: $color-primary !default;
$article-date-font-size: 14px !default;
$article-date-font-weight: bold !default;		

// Article Links - Basic Styling
$article-links-border: 1px solid darken($color-gray-lighten, 10%) !default;

// Article Links - ".left/.right/"
$article-links-link-margin-left: 15px !default;
$article-links-link-margin-right: 15px !default;

// Article Links - padding
$article-links-padding-top: 10px !default;
$article-links-padding-bottom: 10px !default;
$article-links-padding-left: 10px !default;
$article-links-padding-right: 10px !default;

See config.scss for complete reference.
