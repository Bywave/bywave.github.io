Icon Images

If you are using images as icon, use a class name ".icon-image" followed by whatever icons you desire.

, , , , , ,


HTML example markup
<i class="icon-image icon-image-home"></i>
<i class="icon-image icon-image-tag"></i>
<i class="icon-image icon-image-cog"></i>
<i class="icon-image icon-image-instagram"></i>
<i class="icon-image icon-image-envelope"></i>
<i class="icon-image icon-image-check"></i>
<i class="icon-image icon-image-error"></i>
Important!

For icon images, use this namespace ".icon-image-(whatever icon image name)". This is to avoid conflicts to other icon/font services.

Use class name ".icon-right" if you want to .

If you want to position the icon to the right, put the icon on the right, and add a class name ".icon-right", this will provide a default margin-left for better spacing.

Important!

Using images needed to have a retina version of images, to make it smooth.

Since we are using percentage width to adjust sizes, we advice to make the images in equal dimension (same width and height), so it will properly stretch when you use different sizes.



SASS Configuration Variables

// Icon Margin
// Icon Margin - Left and Right margins
$icon-margin: 3px !default;

// Icon Width
$icon-width: 22px !default;				

// Icon Circular
// Icon Circular - basic styling						
$icon-circular-dimension: 50px !default; // Width and Height	
$icon-circular-margin-bottom: 5px !default;

// Icon Circular - Colors
$icon-circular-bg: $color-gray-lighten !default;
$icon-circular-color: $color-default !default;

// Icon Circular - Hover
$icon-circular-bg-hover: darken($color-gray-lighten, 15%) !default;
$icon-circular-color-hover: $color-default !default;

// Icon Circular Sizes
$icon-circular-dimension-small: floor(($icon-circular-dimension * 60) / 100) !default;
$icon-circular-dimension-medium: floor(($icon-circular-dimension * 75) / 100) !default;
$icon-circular-dimension-large: floor(($icon-circular-dimension * 125) / 100) !default;

// Icon Circular Font Sizes - for Font Awesome
$icon-circular-font-size-default: 18px !default;
$icon-circular-font-size-small: 12px !default;
$icon-circular-font-size-medium: 14px !default;
$icon-circular-font-size-large: 24px !default;

// Icon Images 
// Icon Images Sizes - the higher percentage the bigger the size.
$icon-circular-image-dimension-default: 37% !default;
$icon-circular-image-dimension-small: 30% !default;
$icon-circular-image-dimension-medium: 35% !default;
$icon-circular-image-dimension-large: 40% !default;

See config.scss for complete reference.
