Navigation

A main functionality of the site. It is consists of different navigation, (Nav, Top Nav, Side Nav and Footer Nav).



Basic Setup

It uses a list to create navigation. Starts by adding a class name to "<ul>.

HTML example markup
<ul class="type of navigation">
	<li><a href="#">...</a></li>
	<li><a href="#">...</a></li>
	<li><a href="#">...</a></li>
	<li><a href="#">...</a></li>
</ul>


Nav

The main navigation of the site. Use a class name ".nav". Commonly used for navbar.

HTML example markup
<ul class="nav">
	<li class="active"><a href="#">Menu 1</a></li>
	<li><a href="#">Menu 2</a></li>
	<li><a href="#">Menu 3</a></li>
	<li><a href="#">Menu 4</a></li>
</ul>
Important!

For proper alignment, use utility class name's ".left" and ".right".
