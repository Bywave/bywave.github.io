Tables

This is the easiest way to do tabular data and there are so many various styles when designing a tabular data. These sets of style will make it easier for you to adopt any designs.



Basic Setup

A usual setup of Tables, use a tag name "<table>" followed by "<tr>", "<td>" and "<th>".

Important!

Tables can't be controlled in small screens when you have multiple data. So to have a decent looking layout, use a class name ".table-flexible" to have an option to scroll in small screens. It will be followed by "<table>"


HTML example markup
<div class="table-flexible">
	<table>
		<tr>
			<th>TABLE HEADER 1</th>
			<th>TABLE HEADER 2</th>
			<th>TABLE HEADER 3</th>
		</tr>
		<tr>
			<td>Division 1</td>
			<td>Division 2</td>
			<td>Division 3</td>
		</tr>
	</table>
</div>

SASS Configuratoin Variables

// Table basic styling
$table-bg: transparent !default;
$table-color: inherit !default;

// Table padding/margin styling
$table-padding-top: 10px !default;
$table-padding-bottom: 10px !default;
$table-padding-left: 10px !default;
$table-padding-right: 10px !default;

$table-margin-bottom: $margin-bottom-default !default;

// Table border in between
$table-border-between: 2px solid $color-white !default;

// Table th/td vertical alignment
$table-vertical-align: middle !default;

// Table th styling
$table-th-bg: $color-primary !default;
$table-th-color: $color-white !default;
$table-th-font-weight: bold !default;

// Table tr odd/even colored styling
$table-row-odd-bg: $color-gray-lighten !default;
$table-row-odd-color: $color-black !default;
$table-row-even-bg: $color-white !default;
$table-row-even-color: $color-black !default;

// Table tr - hover
$table-row-even-bg-hover: none !default; // darken($table-row-even-bg, 15%)
$table-row-odd-bg-hover: none !default; // darken($table-row-odd-bg, 15%)

// Table w/ borders
$table-border-top: 1px solid darken($color-gray-lighten, 10%) !default;
$table-border-bottom: 1px solid darken($color-gray-lighten, 10%) !default;
$table-border-left: 1px solid darken($color-gray-lighten, 10%) !default;
$table-border-right: 1px solid darken($color-gray-lighten, 10%) !default;

See config.scss for complete reference.
