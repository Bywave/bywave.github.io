Breadcrumbs

To determine what pages you were in, or to access the pages hierarchy.



Basic Setup

Use a class name ".breadcrumbs" and you are good to go. Add a class name ".active" to highlight the link. You can change the separators under "li:after".


  • Home
  • Menu 1
  • Menu 2
  • Menu 3
  • Menu 4


HTML example markup
<ul class="breadcrumbs">
	<li><a href="#">Home</a></li>
	<li><a href="#">Menu 1</a></li>
	<li><a href="#">Menu 2</a></li>
	<li><a href="#">Menu 3</a></li>
	<li class="active"><a href="#">Menu 4</a></li>
</ul>


Breadcrumbs with Background

An optional element, to enhance a layout. Add a class name ".with-bg". Since we put a backgorund, we needed to apply a padding for better spacing.

// Breadcrumbs Basic Styling
$breadcrumbs-color: $color-default !default;
$breadcrumbs-font-size: 14px !default;
$breadcrumbs-margin-bottom: 10px !default;

// Breadcrumbs Padding Styling
$breadcrumbs-padding-top: 10px !default;
$breadcrumbs-padding-bottom: 10px !default;
$breadcrumbs-padding-left: 15px !default;
$breadcrumbs-padding-right:	15px !default;

// Breadcrumbs List - divider
$breadcrumbs-list-divider: '/' !default;
$breadcrumbs-list-divider-color: lighten($color-gray, 30%) !default;

// Breadcrumbs Link - Link
$breadcrumbs-link-color: $color-default !default;

// Breadcrumbs Link - Hover
$breadcrumbs-link-color-hover: $color-primary !default;
$breadcrumbs-link-text-decoration-hover: underline !default;

// Breadcrumbs Link - Active
$breadcrumbs-link-color-active: $color-primary !default;

// Breadcrumbs Background
// Breadcrumbs Background - Styling
$breadcrumbs-bg-list-divider-color: lighten($color-primary, 10%) !default;
$breadcrumbs-bg-bg: $color-primary !default;

// Breadcrumbs Background - Link
$breadcrumbs-bg-color: $color-white !default;

// Breadcrumbs Background - Hover
$breadcrumbs-bg-color-hover: inherit !default;

// Breadcrumbs Background - Hover
$breadcrumbs-bg-color-active: darken($color-primary, 30%) !default;
