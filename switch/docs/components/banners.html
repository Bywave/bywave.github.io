Banners

A unique component for advanced style of images.



Basic Setup

Put all stuffs you need inside an image, including text, buttons, ribbons, tags, etc in a class name ".banner". All the images inside the banner have a default width of 100%.

Ribbon

ribbon

Button

Regular Ribbon with editable text. You can style it on whatever you desire. Image as ribbon can also be used, just put an "<img>" tag or "background".

If you use an image, just drop the image in a class name ".ribbon", no need for ".ribbon-content" (this will only be used if you want an editable text.) class.

Default is "left", but you can change it by adding a class name ".right", ".bottom.right" and ".bottom" (left).


HTML example markup
<div class="banner">
	<div class="banner-image">
		<img src="..." alt="..." />
		<span class="ribbon"><em class="ribbon-content">...</em></span>
	</div>
	<div class="banner-content">
		<p>...</p>
		<a href="#" class="button primary">...</a>
	</div>
</div>

Button

Regular Tag with editable text. Like the ribbon, image can also be used, just put an "<img>" tag or "background"

Default is "left". Use a class name ".tag", and put whatever text you want. This can be also positioned in different location, just add a class name ".right", ".bottom.right" and ".bottom".


HTML example markup
<div class="banner">
	<div class="banner-image">
		<img src="..." alt="..." />
		<span class="tag">...</span>
	</div>
	<div class="banner-content">
		<p>...</p>
		<a href="#" class="button primary">...</a>
	</div>
</div>

SASS Configuration Variables

// Banner Content - padding default
$banner-content-padding-top-default: 15px !default;
$banner-content-padding-bottom-default: 15px !default;
$banner-content-padding-right-default: 15px !default;
$banner-content-padding-left-default: 15px !default;

// Banner Content - padding for small screen
$banner-content-padding-top-small: inherit !default;
$banner-content-padding-bottom-small: inherit !default;
$banner-content-padding-right-small: inherit !default;
$banner-content-padding-left-small: inherit !default;

// Banner Content - padding for medium screen
$banner-content-padding-top-medium: inherit !default;
$banner-content-padding-bottom-medium: inherit !default;
$banner-content-padding-right-medium: inherit !default;
$banner-content-padding-left-medium: inherit !default;

// Ribbon styling
$banner-ribbon-bg: $color-primary !default;
$banner-ribbon-color: $color-white !default;

// Tag styling
$banner-tag-bg: $color-primary !default;
$banner-tag-color: $color-white !default;
$banner-tag-offset: 20px !default; // Position top/bottom offset.

// Banner Caption - padding default
$banner-caption-padding-top-default: 15px !default;
$banner-caption-padding-right-default: 15px !default;
$banner-caption-padding-bottom-default: 15px !default;
$banner-caption-padding-left-default: 15px !default;

// Banner Caption - padding for small screen
$banner-caption-padding-top-small: inherit !default;
$banner-caption-padding-right-small: inherit !default;
$banner-caption-padding-bottom-small: inherit !default;
$banner-caption-padding-left-small: inherit !default;

// Banner Caption - padding for medium screen
$banner-caption-padding-top-medium: inherit !default;
$banner-caption-padding-right-medium: inherit !default;
$banner-caption-padding-bottom-medium: inherit !default;
$banner-caption-padding-left-medium: inherit !default;

// Caption positioning
$banner-caption-position: absolute !default;
$banner-caption-position-top: auto !default;  // value "auto" needed if you want to position to bottom.
$banner-caption-position-bottom: 0 !default; // value "auto" needed if you want to position to top.

// Caption styling - background, color, etc.
$banner-caption-bg: $color-tertiary !default; // rgba($color-primary, 0.5) - for transparent bg.
$banner-caption-color: $color-default !default;
$banner-caption-link-color: $text-link !default;

See config.scss for complete reference.
