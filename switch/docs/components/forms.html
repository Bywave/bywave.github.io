Forms

Styling a form elements, sometimes is annoying espcially on alignment to each other, the labels position or whatsoever. Whether we like it or not, forms always exist on the site. Whether it's newsletter, contact forms, etc. Switch help you style the form in a flexible ways to achieve great results.



Basic Setup

Wrapping your form elements into tag "<form>" with a class of ".form" and form elements inside. All form elements have a default "100%" width, so if you don't want a full width, just give it a container and put the width you desire.

Important!

For Input Text and Labels, use a class name ".input-text" and ".input-label"

Textarea and Select box have a default styles, so no need to add class name.



Simple Form

For a block of form elements like (input, buttons, etc.), just use a class name ".form-category". This will serve a container for better margin to each block.


HTML example markup
<form class="form">
	<div class="form-category">
		<label class="input-label">...</label>
		<input type="text" class="input-text" placeholder="..."/>
	</div>
	<div class="form-category">
		<label class="input-label">...</label>
		<input type="text" class="input-text" placeholder="..."/>
	</div>
</form>


Form Responsive

These Form Elements is now easy to implement, because it is using our very own Grid System. Since it is using our grid system, so basically it is Responsive.

		</div>
	</div>
	<div class="row">
		<div class="grid-6">
			<label class="input-label">...</label>
			<input type="text" class="input-text" placeholder="..."/>
		</div>
		<div class="grid-6">
			<label class="input-label">...</label>
			<input type="text" class="input-text" placeholder="..."/>
		</div>
	</div>
	<button class="button primary right" />
</form>



Form with Inline Elements

For Inline Elements, labels are aligned vertically. So add a class name ".valign" and wrap your text in "<span>" tag to make it work.



HTML example markup
<form class="form">
	<div class="row">
		<div class="grid-2">
			<label class="input-label valign"><span>...<span></label>
		</div>
		<div class="grid-4">
			<input type="text" class="input-text" placeholder="..."/>
		</div>
		<div class="grid-4">
			<input type="text" class="input-text" placeholder="..."/>
		</div>
		<div class="grid-2">
			<button class="button primary full right" />
		</div>
	</div>
	<div class="row">
		<div class="grid-6">
			<div class="row">
				<div class="grid-6">
					<label class="input-label valign"><span>...<span></label>
				</div>
				<div class="grid-6">
					<input type="text" class="input-text" placeholder="..."/>
				</div>
			</div>
		</div>
		<div class="grid-6">
			<div class="row">
				<div class="grid-4">
					<label class="input-label valign"><span>...<span></label>
				</div>
				<div class="grid-5">
					<input type="text" class="input-text" placeholder="..."/>
				</div>
				<div class="grid-3">
					<button class="button primarys right" />
				</div>
			</div>
		</div>
	</div>
</form>

Textarea

For height styling, just adjust the "rows" attribute.


HTML example markup

<textarea rows="10" placeholder="..."></textarea>

Select

This is a regular select box. Wrap your select box in class name ".select-box", followed by "<select>".

The arrow image can be replaced under the class name ".select-box".


HTML example markup
<div class="select-box">
	<select>
		<option>...</option>
	</select>
</div>

Input Category

Adding a flexible input together with labels, buttons, etc. End to end styling of element. For Text, checkbox, etc. Use "<label>" tag.


HTML example markup
<div class="input-category">
	<input type="text" class="input-text" placeholder="..."/>
	<div class="input-append">
		<button class="button primary" />
	</div>
</div>
<div class="input-category">
	<input type="text" class="input-text" placeholder="..."/>
	<div class="input-append">
		<label><input type="radio" name="radio" /></label>
	</div>
</div>

Form with Validation

States for input element is also included, like error and completed. Use a class name ".error" for labels and input text.


HTML example markup
<label class="input-label error">...</label>
<input type="text" class="input-text error" placeholder="..."/>
<label class="input-label completed">...</label>
<input type="text" class="input-text completed" placeholder="..."/>


SASS Configuration Variables

// Form Margin Bottom
$form-margin-bottom: 15px !default;

// Input Padding 
$input-padding-top-default: 7px !default;
$input-padding-bottom-default: 7px !default;
$input-padding-left-default: 10px !default;
$input-padding-right-default:10px !default;

// Input Padding - for medium screens
$input-padding-top-medium: 4px !default;
$input-padding-bottom-medium: 4px !default;
$input-padding-left-medium: 10px !default;
$input-padding-right-medium: 10px !default;

// Input Padding - for small screens
$input-padding-top-small: 4px !default;
$input-padding-bottom-small: 4px !default;
$input-padding-left-small: 10px !default;
$input-padding-right-small: 10px !default;

// Input (text, textarea) - Styling
$input-bg: $color-white !default;
$input-border: 1px solid darken($color-gray-lighten, 10%) !default;
$input-border-radius: 0px !default;
$input-placehoder: $color-gray !default;
$input-color: $color-default !default;

// Input - Font styling
$input-font-family: $font-family-default !default;
$input-transform: none !default;
$input-font-weight: normal !default;

// Input Font-size
$input-font-size-default:14px !default;
$input-font-size-small: 12px !default;
$input-font-size-medium: 13px !default;

// Input height
$input-height: floor(($input-padding-top-default + $input-padding-bottom-default) + ($input-font-size-default * $line-height-default) + 2) !default;

// Input Margin Bottom
$input-margin-bottom: 6px !default;

// Input Shadow Inset
$input-shadow-inset-color: $color-gray !default;
$input-shadow-inset-opacity: 0.2 !default;
$input-shadow-inset-category: 0 1px 1px 0 !default; // h-shadow, v-shadow, blur, spread;

// Input Shadow Outset
$input-shadow-color: darken($color-gray, 10%) !default;
$input-shadow-opacity: 0 !default; // Disable Outset Shadow, by giving 0 opacity;
$input-shadow-category: 0 1px 2px 0 !default; // h-shadow, v-shadow, blur, spread;

// Input Focus
$input-shadow-color-focus: $color-primary !default;

// Label - Styling
$label-font-family: $font-family-primary !default;
$label-margin-bottom: 2px !default;
$label-font-size: 13px !default;
$label-font-weight: bold !default;

// Input Category
$input-category-label-bg: $color-primary !default;
$input-category-label-color: $color-white !default;
$input-category-label-border: 1px solid transparent !default;

// Select Box - Styling
$select-color: 						#7f7f7f !default;

See config.scss for complete reference.
