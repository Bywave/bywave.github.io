Images

To make a website more interesting, images is one of the key factor. And these images also need styling, so we have set of styles to achieve the diserable looks. All images are Responsive in default.



Basic Setup

Important!

All images with classes are displayed as "block" element and also have default "margin-bottom".


Image (default)

Common Image. Can be styled with border, border radius, etc. Use class name ".image".

Image Border

Image with border, can be styled with or without border. Use class name ".border".

Image Circular

Circular image, can be styled with or without border. Use class name ".circular".

Image Thumbnail

Thumbnail image, can be styled in whatever style you desire. Use class name ".thumbnail".


HTML example markup
<img src="..." alt="..." class=image />
<img src="..." alt="..." class=image border />
<img src="..." alt="..." class=image circular />
<img src="..." alt="..." class=image thumbnail />


Flexible Images

Image Full

Image with 100% width. Use class name ".full".


HTML example markup
<img src="..." alt="..." class=image full />

Image Sizes



Image sizes can also be useful. Use class name ".small", ".medium" and ".large".


HTML example markup
<img src="..." alt="..." class=image small />
<img src="..." alt="..." class=image medium />
<img src="..." alt="..." class=image large />


SASS Configuration Variables

// Image - Default basic styling
$image-margin-bottom: $margin-bottom-default !default;
$image-border-radius: 0px !default; 

// Image Border
// Image Border - basic styling
$image-border-border: 1px solid darken($color-gray-lighten, 10%) !default;

// Image Thumbnail
$image-thumbnail-border: $image-border-border !default;
$image-thumbnail-bg: $color-white !default;
$image-thumbnail-padding: 5px !default;

// Image Sizes - small, medium, large
$image-size-small: 25% !default;
$image-size-medium: 40% !default;
$image-size-large: 60% !default;

See config.scss for complete reference.
